como criar um repositório
- git init --> Inicia um repositório
- git status --> oque tem no repositório
- pwd --> ver em que repositorio estamos
- git add documento --> adiciona o documento selecionado para staging area, e em caso de modificação do documento será mostrado na tela
-git rm --cached ."nomedodocumento" --> remove o documento do staging area
- git restore --> volta o arquivo como estava anteriormente, antes da modificação
